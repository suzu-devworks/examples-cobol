# examples-cobol

## What is this repository?

This repository explores COBOL through small, focused programs in the `examples`
collection. It records practical experiments and reference implementations that
demonstrate language features, common patterns, and interoperability-oriented
workflows.

Like other repositories in the collection, it is a public knowledge base rather
than a complete solution set. The examples reflect current understanding and are
expected to evolve over time.

## What topics are covered?

- Core COBOL syntax and control flow basics
- Input/output and interactive console handling
- Data definitions with copybooks and shared records
- Sorting patterns, including legacy and reusable approaches
- Sub-programming patterns (contained, nested, and static calls)
- Data generation examples and UUID-oriented helper flows
- HTTP client usage from COBOL programs
- Build organization with Makefiles across multiple example modules
